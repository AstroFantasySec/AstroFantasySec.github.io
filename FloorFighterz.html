This Alpha built game was created for my 2nd game project
for my unit7 course work for Level 3 UAL Game design & animation
Diploma when I was at WestKings College. This was first pixel game
built I have created on Unreal engine, The end resualt for the experience
was not I expected I was kind of dissapointed. The theme of the Unit was 2d retro games,
We went over retro Console retro , systems etc. We had to Create Game design
document(GDD) of the game we're going to make , mine was a platformer shooter/hackNslash type
game. Then we had to create the sprite design , which wasnt easy The platform sprites like
the floor , objects etc werent that hard. But the Character was hard with some tries I was able to
two a Male and female cop but I was only able to use the male cop in the build. I also had to create some
sprite animation design like Idle , movement, combat and jump. When Implementing what I created
what I created on Unreal engine , while following my old teacher video he created
it was too complex but I was able to do it. The biggest pain in the a** was that I had to quickly
get it done just to start my FMP becuase we had to put it on hold for work experience while all
the other groups theirs and I blame their incompentence to an extent that I couldnt experience it properly
and didnt learn that much on it, that I had to rush on it . Which ruined the experience.
Beside that I have an ok alpha built.

Quick Summery of the game is that you as the cops have to go the abandoned house because of some police
report. And you have investigae to see the issue ...... The full plot summary is on my GGD
If you want to check it out.
