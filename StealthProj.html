Escape The Zone or "ETZ", is a game project I create for my 2nd Final Major project.
This is Beta built version so this not the final build.And I was able to create one level to
show some gameplay.There were some function that was meant to be included in this built but
sadly was removed because of deadline and issue with of the function itself.This game was built
on Unreal 5 on my schools newest computers when we moved to our new building during this year.

This is a stealth game about guy from Texas works a 9 to 5 in day but works for secret organaztion
at night working on cool expensive tech but he gets kidnapped by somepeople who hijacked event he was in
and sent to place where the enemies reside, not knowing his location and now he needs to escape without getting
caught.For the FMP I wanted this game to be simple but fun gameplay and intreseting plot.The game enviroment was
created on Unreal but some of the assets was created Maya and some textures from Photoshop/Unreal.The UI was
made on Photopea(WebPhotoshop) and the character model and animation from Mixamo.

The way this game to be is that this game was thought up before my FMP2. I thought up to create
stealth game on near end of christmas my class and I decided to use our Project 2(which we start at the begining of Jan)
to use it to research , create GDD and test game function before we start implemeted in our FMP kind of like a pre Alpha/Alpha
stage. Which I liked it because it somewhat prepared and ready me for the project.It took me whole January until I started my FMP
on Feb. The influeced I had in the game from stealth games and some stealth games segement, like republique,Assassin creed,
Watch dogs,FFXV(stealth segement),CoD(stealth segement) and Hitman. Some of those game I was currently playing so the influenced
helped and I wanted to make a game that was different to my classmates and stand out so I opt to this idea.

This whole experience was fun and intreseting, I would say this was a somewhat step-up from my last game, there were some challenges
that I came across especially near to the end but all in all it was good experience and less stress from last time.
